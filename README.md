# 🖋️ Scriptora AI  
### Autonomous Multi-Agent Research & Writing System  

Scriptora AI is a professional AI-powered research and writing assistant built using CrewAI and Streamlit.  
It leverages a multi-agent architecture to generate high-quality, research-driven articles based on user input.

---

## 🚀 Features

- 🔍 Multi-Agent Research System (Researcher + Writer)
- 🧠 Deep Topic Analysis
- ✍️ Professionally Structured Article Generation
- 🎨 Premium SaaS-Style UI
- 📄 Downloadable Professionally Formatted PDF
- 👨‍💻 Developer Portfolio Ready Interface

---

## 🏗️ Architecture

Scriptora AI uses:

- **CrewAI** for multi-agent orchestration
- **Gemini LLM (via HuggingFace API)**
- **Streamlit** for frontend UI
- **ReportLab** for PDF generation

### 🧠 Agents

1. **Researcher Agent**
   - Conducts deep topic research
   - Generates structured research summary

2. **Writer Agent**
   - Converts research into a well-written article
   - Ensures readability and clarity
